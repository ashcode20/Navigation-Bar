Welcome!

A Navigation Bar is very important in a website. It's helps in getting to each and every content in the website.
To create a website Navigation Bar you need
1.HTML
2.CSS.


Procedure:

1. First we need to declare the document using the doctype.html tag.
2. Add the HTML tag.
3. Add the head tag.
4. Close the head tag.
5. If you are using an external stylesheet link the css file in the html fie using href tag.
6. Open the body tag
7. Open the header tag. 8. Use the nav tag.
9. Under the nav tag use the ul tag and give the class as nav_links.
10. Under this give a li tag and to this add a href tag and assign the href tag to # and type Home.
11. Close the href and the li tag.
12. Do the same type posts and About us before closing the href and li tag.
13. Close the ul tag and the nav tag.
14. Add a button tag saying contact us.
15. close the header tag.
16. Go to the css file import the font that you want from google fonts.
17. Add the styling for the background the fonts.
18. Add the styling to the head, the subhead and the paragraph.
19. Add the styling to the button and the nav_links.
